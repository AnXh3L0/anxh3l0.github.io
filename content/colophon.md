This website is built with care using modern, privacy-respecting technologies.

## Tech Stack

### Framework
- **[Hugo](https://gohugo.io)** - The world's fastest static site generator
- **[Tailwind CSS](https://tailwindcss.com)** - Utility-first CSS framework

### Hosting & Infrastructure
- **[GitHub Pages](https://pages.github.com)** - Free, reliable static hosting
- **[Cloudflare](https://cloudflare.com)** - DNS and CDN

### Typography
- **[Inter](https://rsms.me/inter/)** - A typeface carefully crafted for computer screens

### Features
- **No tracking** - No Google Analytics or other trackers
- **No cookies** - Preferences stored locally in your browser
- **Webmentions** - IndieWeb standard for interactions
- **RSS feeds** - Subscribe to posts via RSS

### Tools Used
- **[Cursor](https://cursor.com)** - AI-powered code editor
- **[VS Code](https://code.visualstudio.com)** - When I need a break from AI
- **[Git](https://git-scm.com)** - Version control

### Design Principles
1. **Performance first** - Fast load times, minimal JavaScript
2. **Accessibility** - WCAG compliant, keyboard navigable
3. **Privacy** - No third-party tracking
4. **Sustainability** - Lightweight pages = less energy
